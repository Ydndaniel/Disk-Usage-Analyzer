# Disk Usage Analyzer

### Why I built this

I wanted an easier way to explore what was taking up space on my computer without manually searching through mountains of folders.

This project is an Electron desktop application that allows users to browse their file system through a graphical interface. It is currently a work in progress

### Current Features
* Browse local drives
* Navigate folders and subfolders
* Desktop application built with Electron
* JavaScript/Node.js application

### Current Limitations
* File size calculations are currently inaccurate
* Storage visualization is still in development
* File management features have not yet been implemented

### Technologies
* JavaScript
* Node.js
* Electron

### How to Run This Project Locally
* Install Node.js
* git clone https://github.com/Ydndaniel/Disk-Usage-Analyzer.git
* cd Disk-Usage-Analyzer
* npm install
* npm start
