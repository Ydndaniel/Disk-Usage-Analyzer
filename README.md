### Overview

I built this project to make it easier to browse my storage and identify files that could be cleaned up without manually searching through folders.

The application is built with Electron and JavaScript and is currently under active development.

### Current Features
* Browse local drives through a graphical interface
* Navigate folders and subfolders
* Desktop application built with Electron
* Cross-platform project structure using Node.js

### Current Limitations
* File size calculations are currently inaccurate
* Storage visualization is still in development
* File management features have not yet been implemented
